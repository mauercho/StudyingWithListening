# Keep

- 현재 만족하고 있는 부분
  - 논의를 통해 목적에 맞게 논리적인 기획들이 나오고 있음
- 계속 이어갔으면 하는 부분
  - 기획 논의를 지속하여 목표에 맞는 기획 설정

# Problem

- 불편하게 느끼는 부분
  - 눈에 띄면서 사람들에게 도움이 되는 기획을 찾기가 힘듬
- 개선이 필요하다고 생각되는 부분
  - 회의 시 의견 전달이 더 효율적으로 이루어져야 함

# Try

- Problem에 대한 해결책
  - 기존 기획처럼 사회문제를 생각한 후 이를 화제성있게 만들 방법 생각하기
- 다음 회고 때 판별 가능한 것
  - 목적의식에 맞게 고민해서 아이디어 내기
- 당장 실행 가능한 것
  - 아이디어 지속적으로 고민하기
