# Keep

- 현재 만족하고 있는 부분
  - 명확한 목표(문제의식 - 해결방안)를 가지고 기획회의 진행
- 계속 이어갔으면 하는 부분
  - 지속적으로 논리적인 기획 만들어서 의견 나누기

# Problem

- 불편하게 느끼는 부분
  - 기획 내용이 와닿지 않음 / 회의에 대한 집중력이 떨어짐
- 개선이 필요하다고 생각되는 부분
  - 회의 시 의견 전달이 더 효율적으로 이루어져야 함

# Try

- Problem에 대한 해결책
  - 밖에 나가서 / 10층 가서 회의하는 것도 집중력 측면에서 좋을것같아요(된다면)
- 다음 회고 때 판별 가능한 것
  - 주제 논리적으로 고민해서 아이디어 내기
- 당장 실행 가능한 것
  - 회의를 효율적으로 할 방법 논의하기
